To align the text properly in your React component, you can apply CSS styles directly either using inline styles or by adding a CSS class.

1. Inline Style Approach:

If you want to use inline styles within your render method, you can add textAlign to the <div>:

jsx
class HelloWorld extends React.Component {
  state = { name: 'John' };

  render() {
    return (
      <div style={{ textAlign: 'center' }}>
        Hello {this.state.name}!
      </div>
    );
  }
}

2. CSS Class Approach:

Alternatively, if you prefer to use a CSS class for cleaner code or to separate concerns, you can create a CSS file:

CSS File (e.g., App.css):

css
.center-text {
  text-align: center;
}

Then, import the CSS file and use the class in your component:

React Component:

jsx
import './App.css'; // Assuming your CSS file is named App.css

class HelloWorld extends React.Component {
  state = { name: 'John' };

  render() {
    return (
      <div className="center-text">
        Hello {this.state.name}!
      </div>
    );
  }
}

3. Aligning in Other Ways:

  • Left Aligned: Use textAlign: 'left' or the .left-text class with text-align: left;.
  • Right Aligned: Use textAlign: 'right' or the .right-text class with text-align: right;.
  • Vertically Centered: If you need vertical centering along with horizontal centering, you can add additional styles like display: 'flex' and alignItems: 'center':
jsx
class HelloWorld extends React.Component {
  state = { name: 'John' };

  render() {
    return (
      <div style={{
        display: 'flex',
        justifyContent: 'center',
        alignItems: 'center',
        height: '100vh',
        textAlign: 'center'
      }}>
        Hello {this.state.name}!
      </div>
    );
  }
}

This would center the content both horizontally and vertically on the page.
