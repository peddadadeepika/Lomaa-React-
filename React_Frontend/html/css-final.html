3. How do you apply a background color to a webpage?

Hint:
To apply a background color, you use the `background-color` property.
Example:
`body { background-color: #FFFFFF; }`

4. What property is used to change the text color of an element?

Hint:
The `color` property is used to change the text color of an element.
Example:
`element { color: red; }`

Hint:
The `font-family` property is used to change the font of an element.
Example:
`element { font-family: Arial, sans-serif; }`

6. What is the correct CSS syntax for setting the font size to 16px?

Hint:

`element { font-size: 16px; }`

7. How can you make text bold in CSS?

Hint:
You can make text bold by using the `font-weight` property.
Example:
`element { font-weight: bold; }`

9. What is the default value of the position property in CSS?

Hint:
The default value of the `position` property is `static`.
Example:
`element { position: static; }`

10. Which of the following is used to style hyperlinks?

Hint:
You can style hyperlinks using pseudo-classes like `a:hover`, `a:link`, and `a:visited`.
Example:
`a:link { color: blue; }`
`a:hover { color: red; }`

11. What is the value of the display property for inline elements?

Hint:
Inline elements have the value `inline` for the `display` property.
Example:
`element { display: inline; }`

12. Which of the following properties is used to set the space between the content and the border of an element?

Hint:
The `padding` property is used to set the space between the content and the border of an element.
Example:
`element { padding: 10px; }`

13. How do you center a block element horizontally in CSS?

Hint:
To center a block element horizontally, use `margin: auto;`
Example:
`element { margin: auto; width: 50%; }`

14. What does the box-sizing property do?

Hint:
The `box-sizing` property specifies how the width and height of elements are calculated, including padding and border.
Example:
`element { box-sizing: border-box; }`

15. What is the result of applying display: none to an element?

Hint:
`display: none` removes an element from the document flow, and it does not take up any space.
Example:
`element { display: none; }`

16. Which of these selectors targets all elements with the class "menu"?

Hint:
To select elements by class, use the class selector which is denoted by a period (`.`).
Example:
`.menu { color: red; }`

17. How do you target an element with a specific ID in CSS?

Hint:
To select an element by ID, use the ID selector which is denoted by a hash (`#`).
Example:
`#id-name { color: blue; }`

18. How can you apply styles to a specific group of elements in a single CSS rule?

Hint:
To target multiple elements, separate each element with a comma.
Example:
`div, span, p { color: green; }`

19. Which CSS property is used to control the visibility of an element?

Hint:
The `visibility` property controls whether an element is visible or hidden while still taking up space.
Example:
`element { visibility: hidden; }`

21. Which CSS property defines the transparency of an element?

Hint:
The `opacity` property is used to define the transparency of an element.

`element { opacity: 0.5; }`

22. How can you add multiple background images to a single element in CSS?

Hint:
You can use `background-image` to add multiple background images by separating them with commas.
Example:
`element { background-image: url(image1), url(image2); }`

23. What does the z-index property control in CSS?

Hint:
The `z-index` property controls the stacking order of elements. Elements with a higher `z-index` are stacked above those with a lower `z-index`.
Example:
`element { z-index: 10; }`

24. What is the purpose of CSS Flexbox?

Hint:
Flexbox is a layout model that allows for flexible arrangement and alignment of elements within a container.
Example:
`container { display: flex; }`

25. What is a CSS Grid Layout?

Hint:
CSS Grid Layout is a two-dimensional layout system that allows you to arrange elements into rows and columns.
Example:
`container { display: grid; grid-template-columns: 1fr 1fr; }`

28. Which property defines the distance between the text and the edge of a container?
